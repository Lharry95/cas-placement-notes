# cas-placement-notes

A repository to document my learning, notes, daily progress, and code experiments during my work placement.

## Purpose

The purpose of this repository is to:

- document my weekly learning during work placement

- save useful code snippets for future reference
- show evidence of growth, reflection, and practical experience

- General web development concepts
- Problem-solving and debugging
